# Sprint_6

UI-автотесты для учебного сервиса «Яндекс.Самокат».

Тесты реализованы на Python с использованием Selenium WebDriver и pytest.
Применён паттерн Page Object Model.
Подключена генерация отчёта Allure.

---

## Реализованные сценарии

### 1. Раздел «Вопросы о важном»

Проверяется:

- раскрытие каждого вопроса;
- отображение корректного текста ответа.

Реализован тест с параметризацией, который проверяет раскрытие каждого вопроса и корректность текста ответа.

---

### 2. Позитивный сценарий оформления заказа

Проверяется полный пользовательский флоу:

- нажатие кнопки "Заказать";
- заполнение формы "Для кого самокат";
- заполнение формы "Про аренду";
- подтверждение заказа;
- появление окна успешного создания заказа.

Используются два набора тестовых данных.

---

### 3. Точки входа в оформление заказа

Проверяется запуск сценария:

- через кнопку "Заказать" в верхней части страницы;
- через кнопку "Заказать" в нижней части страницы.

---

### 4. Проверка логотипов

Проверяется:

- клик по логотипу "Самокат" -> переход на главную страницу;
- клик по логотипу "Яндекс" -> открытие новой вкладки и редирект на страницу Дзен.

## Запуск проекта

1. Создать виртуальное окружение:
python -m venv venv

2. Активировать окружение:
macOS / Linux:
source venv/bin/activate
Windows:
venv\Scripts\activate

3. Установить зависимости:
pip install -r requirements.txt

4. Запустить тесты:
pytest -v


## Генерация отчёта Allure

Сохранить результаты:
pytest -v --alluredir=allure_results


Открыть отчёт:
allure serve allure_results
